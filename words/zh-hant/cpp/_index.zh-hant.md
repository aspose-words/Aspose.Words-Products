<p><strong>Aspose.Words for C++</strong> 是一種高級 Word 文檔處理 API，用於執行廣泛的處理任務。 Words 的 API 直接在您自己的 C++ 應用程序中提供文檔格式化、操作、郵件合併、水印和更多功能，而無需 Microsoft Word。</p>
<p>Aspose.Words API 支持大多數 Microsoft Word 格式，並且可以將文檔頁面呈現為多種圖像格式。因此，您可以使用 DOC、DOCX、PDF、RTF、DOT、DOTX、DOTM、DOCM、FlatOPC、ODT、OTT、HTML、MHTML、MOBI、TXT、XPS、PS、TIFF、JPEG、PNG、BMP、 SVG、EMF、GIF、PCL、EPUB、XML、XAML 等。 你可以看到 <a href="https://docs.aspose.com/words/cpp/supported-document-formats/">支持格式的完整列表</a>.</p>
<h2 class="h2title">Functionality of Aspose.Words for C++</h2>
<p>Aspose.Words 是一個獨立的 C++ 庫。您無需安裝其他應用程序即可使您的 C++ 代碼正常工作。 這個 C++ API 依賴於 <a href="https://docs.aspose.com/words/cpp/aspose-words-document-object-model/">文檔對像模型 (DOM)</a> 訪問文檔節點和元素。</p>
