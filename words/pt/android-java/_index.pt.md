<p>
 Aspose.Words para Android via Java é uma API de processamento de documentos que permite realizar tarefas de manipulação de documentos diretamente em seus aplicativos Android. A API permite criar, manipular, converter e renderizar facilmente arquivos do Microsoft Word.
</p>

<p>
 O Aspose.Words para Android via Java suporta o processamento de formatos nativos como DOC, DOCX, OOXML, TXT, RTF, bem como muitos outros formatos comumente usados, como HTML, OpenDocument ODT, PDF, EPUB, XPS e formatos de imagem como JPEG, PNG , SVG e EMF.
</p>

   <div class="col-lg-12">
    <h2 class="h2title">
     Conversão de formatos de arquivo
    </h2>
    <p>
     A capacidade de converter arquivos diferentes de forma rápida e confiável com alto grau de precisão é o motivo pelo qual a maioria dos nossos clientes escolhe a API. A Android Microsoft Word API permite converter arquivos de qualquer formato compatível para qualquer outro formato compatível, por exemplo, converter qualquer documento em SWF com alta fidelidade e o SWF gerado contém as páginas do documento e um visualizador incorporado, permitindo que o documento seja visualizado em qualquer dispositivo. Os desenvolvedores podem converter formatos com facilidade usando apenas duas linhas de código.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Salvar documento em diferentes formatos - Java
     </h3>
     <pre><code class="java">// carrega o arquivo a ser convertido

Document doc = new Document(SD_PATH + "template.doc");

// Converte DOC para DOCX, PDF e HTML

doc.save(SD_PATH + "output.docx", SaveFormat.DOCX);

doc.save(SD_PATH + "output.pdf", SaveFormat.PDF);

doc.save(SD_PATH + "output.html", SaveFormat.HTML);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Suporte para OOXML, Flat OPC e Word 2003 XML
    </h2>
    <p>
     Essa biblioteca oferece amplo suporte para carregar, visualizar, salvar e converter arquivos WordprocessingML. Essa implementação abrangente é possível porque a API projetada com a estrutura dos documentos do Microsoft Word em mente (e o WordprocessingML é conhecido por imitar a representação interna dos arquivos do Microsoft Word), simplificando a inclusão de suporte para essas implementações em nosso mecanismo de produto principal.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Carregar e salvar formatos HTML
    </h2>
    <p>
     A importação de HTML e XHTML das APIs do Android Word é resiliente a documentos que não são bem formados. Elementos ou atributos inválidos ou não reconhecidos são ignorados com segurança durante o processo de importação, enquanto a conversão para os formatos HTML e XHTML é personalizável. Muitos parâmetros como a exportação de propriedades de documentos, cabeçalhos e rodapés, localizações de imagens, dimensionamento e resolução, opções de exportação de campos de formulário e outros podem ser especificados.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Manipulação com modelo de objeto de documento
    </h2>
    <p>
     A biblioteca de documentos do Android permite que os desenvolvedores criem, construam e modifiquem arquivos - incluindo formatação com um modelo de objeto de documento intuitivo que consiste em mais de 100 classes.
    </p>
    <p>
     Os desenvolvedores do Android podem criar, modificar, extrair e substituir programaticamente todos os elementos, incluindo seções, cabeçalhos, rodapés, parágrafos, listas, tabelas, texto, campos, hiperlinks, marcadores e imagens. A API também permite especificar formatação detalhada para objetos de desenho, incluindo formas, caixas de texto, imagens, objetos OLE e controles.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Relatórios e mala direta
    </h2>
    <p>
     Você pode usar a API Android Microsoft Word como uma solução de relatórios completa. Crie relatórios no Microsoft Word e permita que a API preencha documentos com dados de várias fontes.
    </p>
    <p>
     A API permite que os desenvolvedores usem campos de mala direta padrão do Microsoft Word em relatórios. Além disso, você pode usar a sintaxe estendida das APIs para campos de mala direta para realizar operações mais complexas durante operações de mala direta, como inserir imagens ou regiões repetíveis. Os dados que orientam seus relatórios e mala direta podem vir de objetos ResultSet, objetos de negócios ou de uma fonte de dados totalmente personalizada.
    </p>
