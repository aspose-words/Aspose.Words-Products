<p>
 Aspose.Words for SharePoint torna possível converter e combinar documentos do Word dentro de aplicativos do Microsoft SharePoint. Ele suporta conversão de formato com alta fidelidade, além de concatenar vários arquivos durante a conversão. A API pode ser usada para gerar relatórios com base em um documento de modelo do Microsoft Word com dados de bancos de dados SQL, arquivos XML ou listas do SharePoint.
</p>

     Different Opções de conversão
    </h2>
    <p>
     Aspose.Words for SharePoint permite concatenar arquivos, converter arquivos em subpastas, alterar opções de pasta de destino, substituir arquivos existentes em pastas de destino e navegar na pasta de destino durante a conversão. Além disso, o aplicativo permite combinar qualquer número de arquivos de entrada (em formatos suportados) em um único documento de saída. Você pode pegar arquivos de subpastas na biblioteca de documentos, escolher os arquivos desejados, ajustar a sequência de concatenação por meio de uma interface simples e obter um arquivo combinado resultante com um número mínimo de cliques.
    </p>
   </div>
   <div class="col-lg-12">

     Reporting Features
    </h2>

    </p>
    <p>
     Um arquivo do Microsoft Word serve como um modelo de relatório com espaços reservados definindo onde inserir dados da fonte de dados do relatório. Os campos regulares de mala direta do Microsoft Word servem como espaços reservados com muitas possibilidades, como:
    </p>
    <ul class="unstyled">
     <li>
      Regiões para repetir o registro de dados em uma fonte de dados
     </li>
     <li>
      Regiões aninhadas para refletir as relações pai-filho nos dados
     </li>
     <li>
      Espaços reservados de imagem para inserir imagens de uma fonte de dados
     </li>
    </ul>

É uma alternativa perfeita de automação do Microsoft Word em termos de recursos suportados, segurança, estabilidade, escalabilidade, velocidade e preço.
