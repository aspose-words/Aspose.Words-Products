<p><strong>Aspose.Words for C++</strong> é uma API avançada de processamento de documentos do Word para executar uma ampla variedade de tarefas de processamento. A API do Words fornece formatação de documentos, manipulação, mala direta, marca d'água e muito mais recursos diretamente em seus próprios aplicativos C++ sem exigir o Microsoft Word.</p>
<p>A API Aspose.Words suporta a maioria dos formatos do Microsoft Word e pode renderizar as páginas do documento para vários formatos de imagem. Assim, você pode trabalhar com documentos em DOC, DOCX, PDF, RTF, DOT, DOTX, DOTM, DOCM, FlatOPC, ODT, OTT, HTML, MHTML, MOBI, TXT, XPS, PS, TIFF, JPEG, PNG, BMP, SVG, EMF, GIF, PCL, EPUB, XML, XAML e muito mais. Você pode ver <a href="https://docs.aspose.com/words/cpp/supported-document-formats/">a lista completa de formatos suportados</a>.</p>
<h2 class="h2title">Functionality of Aspose.Words for C++</h2>
<p>Aspose.Words é uma biblioteca C++ independente. Você não precisa instalar aplicativos adicionais para que seu código C++ funcione. Esta API C++ depende do <a href="https://docs.aspose.com/words/cpp/aspose-words-document-object-model/">Modelo de Objeto de Documento (DOM)</a> para acessar nós e elementos do documento.</p>
