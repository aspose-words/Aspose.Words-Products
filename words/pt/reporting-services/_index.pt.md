<p>
 O Aspose.Words for Reporting Services é uma extensão de renderização que permite aos desenvolvedores exportar relatórios RDL e RDLC como arquivos DOC, DOCX, PDF, RTF, HTML, MHTML, ODT, TXT, XPS e WordprocessingML de dentro do Microsoft SQL Server Reporting Services.
</p>

<p>
 Aspose.Words para Reporting Services suporta totalmente as especificações RDL. Você pode usar qualquer designer de relatório para reprojetar seus relatórios existentes ou simplesmente exportar qualquer relatório RDL existente como um documento do Microsoft Word, e ele renderizará exatamente como você esperaria - idêntico ao design RDL.
</p>

    <h2 class="h2title">
     Alta Fidelidade, Idêntico ao Design de Relatório
    </h2>
    <p>
     O Aspose.Words for Reporting Services exporta relatórios para arquivos do Microsoft Word de forma que eles pareçam idênticos ao design original do relatório, conforme produzido pelos renderizadores integrados do Microsoft SQL Server Reporting Services. Você pode escolher entre dois modos de exportação ou experimentar os dois tipos de layout alterando uma única configuração para descobrir qual deles atende melhor às suas necessidades:
    </p>
    <ul class="unstyled">
     <li>
      Layout de fluxo
     </li>
     <li>
      Layout posicionado
     </li>
    </ul>
    <br/>
    <p>
     <strong>
      Layout de fluxo
     </strong>
     produz "editável" documentos que consistem em parágrafos simples, tabelas, seções - isso é semelhante em comparação a um funcionário de escritório criando o documento no Microsoft Word.
    </p>
    <p>
     <strong>
      Layout posicionado
     </strong>
     produz relatórios com a fidelidade de conversão mais precisa de RDL para Microsoft Word. Tabelas e matrizes de relatórios são exportadas como tabelas do Microsoft Word, enquanto alguns elementos são exportados como caixas de texto posicionadas para manter a fidelidade o mais próximo possível do design original.
    </p>
   </div>

É uma alternativa perfeita de automação do Microsoft Word em termos de recursos suportados, segurança, estabilidade, escalabilidade, velocidade e preço.
