<p>
 Aspose.Words for Reporting Services to rozszerzenie renderowania umożliwiające programistom eksportowanie raportów RDL i RDLC jako plików DOC, DOCX, PDF, RTF, HTML, MHTML, ODT, TXT, XPS i WordprocessingML z usług Microsoft SQL Server Reporting Services.
</p>

<p>
 Aspose.Words for Reporting Services w pełni obsługuje specyfikacje RDL. Możesz użyć dowolnego projektanta raportów, aby przeprojektować istniejące raporty lub po prostu wyeksportować dowolny istniejący raport RDL jako dokument programu Microsoft Word, a wyświetli się on dokładnie tak, jak byś tego oczekiwał — identycznie jak projekt RDL.
</p>

    <h2 class="h2title">
     Wysoka wierność, identyczna z projektem raportu
    </h2>
    <p>
     Aspose.Words for Reporting Services eksportuje raporty do plików programu Microsoft Word w taki sposób, że wyglądają identycznie jak oryginalny projekt raportu utworzony przez wbudowane moduły renderujące usług Microsoft SQL Server Reporting Services. Możesz wybrać jeden z dwóch trybów eksportu lub wypróbować oba typy układów, zmieniając jedno ustawienie konfiguracji, aby dowiedzieć się, który z nich najlepiej odpowiada Twoim potrzebom:
    </p>
    <ul class="unstyled">
     <li>
      Układ przepływu
     </li>
     <li>
      Układ pozycjonowany
     </li>
    </ul>
    <br/>
    <p>
     <strong>
      Układ przepływu
     </strong>
     produkuje "edytowalne" dokumenty składające się z prostych akapitów, tabel, sekcji – podobnie jak pracownik biurowy tworzący dokument w Microsoft Word.
    </p>
    <p>
     <strong>
      Układ pozycjonowany
     </strong>
     tworzy raporty z najdokładniejszą wiernością konwersji z RDL do Microsoft Word. Tabele i macierze z raportów są eksportowane jako tabele programu Microsoft Word, a niektóre elementy są eksportowane jako ustawione pola tekstowe, aby zachować wierność jak najbardziej zbliżoną do oryginalnego projektu.
    </p>

Jest to idealna alternatywa automatyzacji Microsoft Word pod względem obsługiwanych funkcji, bezpieczeństwa, stabilności, skalowalności, szybkości i ceny.
