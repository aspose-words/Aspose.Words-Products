<p><strong>Aspose.Words for C++</strong> ist eine fortschrittliche API zur Verarbeitung von Word-Dokumenten, mit der eine Vielzahl von Verarbeitungsaufgaben ausgeführt werden können. Die API von Words bietet Dokumentformatierung, Bearbeitung, Seriendruck, Wasserzeichen und viele weitere Funktionen direkt in Ihren eigenen C++-Anwendungen, ohne dass Microsoft Word erforderlich ist.</p>
<p>Die Aspose.Words-API unterstützt die meisten Microsoft Word-Formate und kann die Dokumentseiten in eine Reihe von Bildformaten rendern. So können Sie mit Dokumenten in DOC, DOCX, PDF, RTF, DOT, DOTX, DOTM, DOCM, FlatOPC, ODT, OTT, HTML, MHTML, MOBI, TXT, XPS, PS, TIFF, JPEG, PNG, BMP, SVG, EMF, GIF, PCL, EPUB, XML, XAML und mehr. Du kannst sehen <a href="https://docs.aspose.com/words/cpp/supported-document-formats/">die vollständige Liste der unterstützten Formate</a>.</p>
<h2 class="h2title">Functionality of Aspose.Words for C++</h2>
<p>Aspose.Words ist eine eigenständige C++-Bibliothek. Sie müssen keine zusätzlichen Anwendungen installieren, damit Ihr C++-Code funktioniert. Diese C++-API basiert auf der <a href="https://docs.aspose.com/words/cpp/aspose-words-document-object-model/">Dokumentobjektmodell (DOM)</a> um auf Dokumentknoten und -elemente zuzugreifen.</p>
