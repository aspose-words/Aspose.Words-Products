<p>
 Aspose.Words for Reporting Services ist eine Rendering-Erweiterung, mit der Entwickler RDL- und RDLC-Berichte als DOC-, DOCX-, PDF-, RTF-, HTML-, MHTML-, ODT-, TXT-, XPS- und WordprocessingML-Dateien aus Microsoft SQL Server Reporting Services exportieren können.
</p>

<p>
 Aspose.Words for Reporting Services unterstützt die RDL-Spezifikationen vollständig. Sie können einen beliebigen Berichtsdesigner verwenden, um Ihre vorhandenen Berichte neu zu gestalten, oder einfach einen beliebigen vorhandenen RDL-Bericht als Microsoft Word-Dokument exportieren, und er wird genau so wiedergegeben, wie Sie es erwarten würden – identisch mit dem RDL-Design.
</p>

    <h2 class="h2title">
     Exportieren Sie Berichte in viele bekannte Formate
    </h2>
    <p>
     Aspose.Words for Reporting Services bietet eine Reihe von Exportformaten. Sie können Ihre Berichte gerne in "native" Microsoft Word-Dokumentformate wie DOC, DOCX, RTF und WordML oder zusätzliche Formate wie ODT, HTML, MHTML exportieren. TXT, XPS und ePub.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Hohe Wiedergabetreue, identisch mit dem Berichtsdesign
    </h2>
    <p>
     Aspose.Words for Reporting Services exportiert Berichte so in Microsoft Word-Dateien, dass sie mit dem ursprünglichen Berichtsdesign identisch erscheinen, wie es von den integrierten Renderern von Microsoft SQL Server Reporting Services erstellt wird. Sie können zwischen zwei Exportmodi wählen oder beide Layouttypen ausprobieren, indem Sie eine einzige Konfigurationseinstellung ändern, um herauszufinden, welche Ihren Anforderungen am besten entspricht:
    </p>
    <ul class="unstyled">
     <li>
      Fluss-Layout
     </li>
     <li>
      Positioniertes Layout
     </li>
    </ul>
    <br/>
    <p>
     <strong>
      Fluss-Layout
     </strong>
     produziert "bearbeitbar" Dokumente, die aus einfachen Absätzen, Tabellen, Abschnitten bestehen - das ist vergleichbar mit einem Büroangestellten, der das Dokument in Microsoft Word erstellt.
    </p>
    <p>
     <strong>
      Positioniertes Layout
     </strong>
     erstellt Berichte mit der genauesten Konvertierungstreue von RDL zu Microsoft Word. Tabellen und Matrizen aus Berichten werden als Microsoft Word-Tabellen exportiert, während einige Elemente als positionierte Textfelder exportiert werden, um die Wiedergabetreue so nah wie möglich am ursprünglichen Design zu halten.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Microsoft Word-Automatisierung – nicht erforderlich
    </h2>
    <p>
     Aspose.Words for Reporting Services basiert auf verwaltetem Code, für den niemals Microsoft Word auf dem Computer installiert sein muss, um mit Word-Dokumentformaten zu arbeiten. Es ist eine perfekte Microsoft Word-Automatisierungsalternative in Bezug auf unterstützte Funktionen, Sicherheit, Stabilität, Skalierbarkeit, Geschwindigkeit und Preis.
