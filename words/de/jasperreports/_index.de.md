<p>
 Aspose.Words for JasperReports ist die einzige Lösung auf dem Markt, die es ermöglicht, Berichte von JasperReports und JasperServer in Microsoft Word-Dokumente (DOC), Office Open XML (OOXML, DOCX), Rich Text Format (RTF), OpenDocument Text ( ODT), Webseiten- (HTML) und Nur-Text-Formate (TXT) ohne Verwendung von Microsoft Word. Alle Berichtsfunktionen mit höchster Genauigkeit können in eine Microsoft Word-Datei konvertiert werden.
</p>
