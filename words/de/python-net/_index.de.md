<p><strong>Aspose.Words for Python via .NET</strong> ist ein Paket, das zum Lesen und Bearbeiten von Dokumenten verschiedener Art entwickelt wurde, mit einem Hauptaugenmerk auf Microsoft Word- (DOC, DOCX, ODT), PDF- und Web-Dokumenten (HTML, Markdown). Unsere Bibliothek enthält mehr als 100 Python-Klassen zur Handhabung von Low-Level-Operationen hinter der Dokumentenverarbeitung und Datenformatierung. Somit bietet Aspose.Words Python-Entwicklern eine einzigartige Gelegenheit, skriptbasierte Dokumentenautomatisierung zu implementieren.</p>
<p>Mit dem Aufkommen leistungsfähiger Computerhardware in den letzten Jahrzehnten hat sich Python zu einer Programmiersprache entwickelt, die in der Webentwicklung (Django-, Flask-, Pyramid-Frameworks), wissenschaftlichen Aufgaben und Datenanalysen weit verbreitet ist. Die Nützlichkeit von Python ergibt sich aus einem riesigen Ökosystem von Paketen von Drittanbietern. Ausgestattet mit einer großen Anzahl sich fortschreitend entwickelnder Bibliotheken, gilt Python als die beste Wahl für die schnelle Softwareentwicklung.</p>
<p>Das Team von Aspose.Words entschied sich, mit den neuesten Programmiertrends Schritt zu halten und die Python-Community mit professioneller Software zur Verarbeitung von DOC, DOCX, PDF, RTF, DOT, DOTX, DOTM, DOCM, FlatOPC, ODT, OTT, HTML, MHTML, MOBI, TXT, XPS, PS, TIFF, JPEG, PNG, BMP, SVG, EMF, GIF, PCL, EPUB, XML, XAML und andere Dateiformate. Du kannst sehen <a href="https://docs.aspose.com/words/python-net/supported-document-formats/">die vollständige Liste der unterstützten Formate</a>.</p>
<p><i>Notiz:</i> laut TIOBE Programming Index, der die Popularität von Programmiersprachen widerspiegelt, ist Python heute eine Programmiersprache Nummer eins.</p>
<h2 class="h2title">Funktionalität von Aspose.Words für Python über .NET</h2>
<p>Aspose.Words ist eine eigenständige Python-Bibliothek. Sie müssen keine Office-Suite installieren, damit Ihr Python-Code funktioniert. Diese Python-API basiert auf der <a href="https://docs.aspose.com/words/python-net/aspose-words-document-object-model/">Dokumentobjektmodell (DOM)</a> um auf Dokumentknoten und -elemente zuzugreifen.</p>

      Erstellen Sie ein Dokument und speichern Sie es als PDF – Python
     </h3>
<pre>
<code class="python">
import aspose.words as aw

# Erstellen Sie ein leeres Dokument
doc = aw.Document()

# Verwenden Sie eine DocumentBuilder-Instanz, um dem Dokument Inhalt hinzuzufügen
builder = aw.DocumentBuilder(doc)

# Fügen Sie dem Dokument einen Absatz hinzu
builder.writeln("Hello World!")

# Speichern Sie das Ergebnis als PDF-Dokument. Das Ausgabeformat wird durch die Dateiendung bestimmt
doc.save("Output.pdf")
</code>
