<p><strong>Aspose.Words for C++</strong> — это расширенный API обработки документов Word для выполнения широкого спектра задач обработки. API Words обеспечивает форматирование документов, манипулирование ими, слияние почты, нанесение водяных знаков и многие другие возможности непосредственно в ваших собственных приложениях C++, не требуя Microsoft Word.</p>
<p>Aspose.Words API поддерживает большинство форматов Microsoft Word и может отображать страницы документа в различных форматах изображений. Таким образом, вы можете работать с документами в форматах DOC, DOCX, PDF, RTF, DOT, DOTX, DOTM, DOCM, FlatOPC, ODT, OTT, HTML, MHTML, MOBI, TXT, XPS, PS, TIFF, JPEG, PNG, BMP, SVG, EMF, GIF, PCL, EPUB, XML, XAML и другие. Ты можешь видеть <a href="https://docs.aspose.com/words/cpp/supported-document-formats/">полный список поддерживаемых форматов</a>.</p>
<h2 class="h2title">Functionality of Aspose.Words for C++</h2>
<p>Aspose.Words — это автономная библиотека C++. Вам не нужно устанавливать дополнительные приложения, чтобы заставить ваш код C++ работать. Этот C++ API основан на <a href="https://docs.aspose.com/words/cpp/aspose-words-document-object-model/">Объектная модель документа (DOM)</a> для доступа к узлам и элементам документа.</p>
