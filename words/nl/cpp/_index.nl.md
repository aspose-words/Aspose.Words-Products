<p><strong>Aspose.Words for C++</strong> is een geavanceerde Word Document Processing API om een breed scala aan verwerkingstaken uit te voeren. De API van Words biedt documentopmaak, manipulatie, mail merge, watermerken en nog veel meer mogelijkheden rechtstreeks binnen uw eigen C++-applicaties zonder dat Microsoft Word nodig is.</p>
<p>Aspose.Words API ondersteunt de meeste Microsoft Word-indelingen en kan de documentpagina's naar een aantal afbeeldingsindelingen weergeven. Zo kunt u werken met documenten in DOC, DOCX, PDF, RTF, DOT, DOTX, DOTM, DOCM, FlatOPC, ODT, OTT, HTML, MHTML, MOBI, TXT, XPS, PS, TIFF, JPEG, PNG, BMP, SVG, EMF, GIF, PCL, EPUB, XML, XAML en meer. Je kan zien <a href="https://docs.aspose.com/words/cpp/supported-document-formats/">de volledige lijst met ondersteunde formaten</a>.</p>
<h2 class="h2title">Functionality of Aspose.Words for C++</h2>
<p>Aspose.Words is een op zichzelf staande C++-bibliotheek. U hoeft geen extra toepassingen te installeren om uw C++-code werkend te krijgen. Deze C++ API is gebaseerd op de <a href="https://docs.aspose.com/words/cpp/aspose-words-document-object-model/">Documentobjectmodel (DOM)</a> om toegang te krijgen tot documentknooppunten en -elementen.</p>
