<p>
 Aspose.Words pour Android via Java est une API de traitement de documents vous permettant d'effectuer des tâches de manipulation de documents directement dans vos applications Android. L'API vous permet de créer, manipuler, convertir et rendre facilement des fichiers Microsoft Word.
</p>

<p>
 Aspose.Words pour Android via Java prend en charge le traitement des formats natifs tels que DOC, DOCX, OOXML, TXT, RTF ainsi que de nombreux autres formats couramment utilisés tels que HTML, OpenDocument ODT, PDF, EPUB, XPS et des formats d'image tels que JPEG, PNG , SVG et EMF.
</p>

   <div class="col-lg-12">
    <h2 class="h2title">
     Conversion de formats de fichiers
    </h2>
    <p>
     La capacité de convertir rapidement et de manière fiable différents fichiers avec un degré élevé de précision est la raison pour laquelle la plupart de nos clients choisissent l'API. L'API Android Microsoft Word vous permet de convertir des fichiers de n'importe quel format pris en charge vers n'importe quel autre format pris en charge, par exemple convertir n'importe quel document en SWF avec une haute fidélité et le SWF généré contient les pages du document et une visionneuse intégrée permettant de visualiser le document sur n'importe quel appareil. Les développeurs peuvent facilement convertir des formats en utilisant seulement deux lignes de code.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Enregistrer le document dans différents formats - Java
     </h3>
     <pre><code class="java">// charge le fichier à convertir

Document doc = new Document(SD_PATH + "template.doc");

// Convertir DOC en DOCX, PDF et HTML

doc.save(SD_PATH + "output.docx", SaveFormat.DOCX);

doc.save(SD_PATH + "output.pdf", SaveFormat.PDF);

doc.save(SD_PATH + "output.html", SaveFormat.HTML);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Compatibilité avec OOXML, Flat OPC et Word 2003 XML
    </h2>
    <p>
     Cette bibliothèque fournit une assistance étendue pour le chargement, l'affichage, l'enregistrement et la conversion de fichiers WordprocessingML. Une telle implémentation complète est possible car l'API conçue avec la structure des documents Microsoft Word à l'esprit (et WordprocessingML est connu pour imiter la représentation interne des fichiers Microsoft Word), ce qui nous permet d'inclure facilement la prise en charge de ces implémentations dans notre moteur de produit principal.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Charger et enregistrer les formats HTML
    </h2>
    <p>
     L'importation HTML et XHTML des API Android Word est résiliente aux documents qui ne sont pas bien formés. Les éléments ou attributs non valides ou non reconnus sont ignorés en toute sécurité pendant le processus d'importation, tandis que la conversion aux formats HTML et XHTML est personnalisable. De nombreux paramètres tels que l'exportation des propriétés du document, les en-têtes et les pieds de page, les emplacements des images, la mise à l'échelle et la résolution, les options d'exportation des champs de formulaire et autres peuvent tous être spécifiés.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Manipulation avec le modèle d'objet de document
    </h2>
    <p>
     La bibliothèque de documents Android permet aux développeurs de créer, de construire et de modifier des fichiers, y compris le formatage avec un modèle d'objet de document intuitif composé de plus de 100 classes.
    </p>
    <p>
     Les développeurs Android peuvent créer, modifier, extraire et remplacer par programme tous les éléments, y compris les sections, les en-têtes, les pieds de page, les paragraphes, les listes, les tableaux, le texte, les champs, les hyperliens, les signets et les images. L'API permet également de spécifier une mise en forme détaillée pour les objets de dessin, y compris les formes, les zones de texte, les images, les objets OLE et les contrôles.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Création de rapports et publipostage
    </h2>
    <p>
     Vous pouvez utiliser l'API Android Microsoft Word comme solution de création de rapports complète. Concevez des rapports dans Microsoft Word, puis autorisez l'API à remplir les documents avec des données provenant de différentes sources.
    </p>
    <p>
     L'API permet aux développeurs d'utiliser des champs de publipostage Microsoft Word standard dans les rapports. De plus, vous pouvez utiliser la syntaxe étendue des API pour les champs de publipostage afin d'effectuer des opérations plus complexes lors d'opérations de publipostage, telles que l'insertion d'images ou de régions répétables. Les données qui pilotent vos rapports et publipostages peuvent provenir d'objets ResultSet, d'objets métier ou d'une source de données entièrement personnalisée.
    </p>
   </div>
