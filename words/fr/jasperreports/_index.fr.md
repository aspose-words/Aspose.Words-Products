 Aspose.Words pour JasperReports est la seule solution sur le marché qui permet d'exporter des rapports de JasperReports et JasperServer vers un document Microsoft Word (DOC), Office Open XML (OOXML, DOCX), Rich Text Format (RTF), OpenDocument Text ( ODT), pages Web (HTML) et texte brut (TXT) sans utiliser Microsoft Word.
