<p>
 Aspose.Words para Android a través de Java es una API de procesamiento de documentos que le permite realizar tareas de manipulación de documentos directamente dentro de sus aplicaciones de Android. La API le permite crear, manipular, convertir y renderizar fácilmente archivos de Microsoft Word.
</p>

<p>
 Aspose.Words para Android a través de Java admite el procesamiento de formatos nativos como DOC, DOCX, OOXML, TXT, RTF, así como muchos otros formatos de uso común como HTML, OpenDocument ODT, PDF, EPUB, XPS y formatos de imagen como JPEG, PNG. , SVG y CEM.
</p>

   <div class="col-lg-12">
    <h2 class="h2title">
     Conversión de formatos de archivo
    </h2>
    <p>
     La capacidad de convertir archivos diferentes de manera rápida y confiable con un alto grado de precisión es la razón por la que la mayoría de nuestros clientes eligen la API. La API de Microsoft Word de Android le permite convertir archivos de cualquier formato compatible a cualquier otro formato compatible, por ejemplo, convertir cualquier documento a SWF con alta fidelidad y el SWF generado contiene las páginas del documento y un visor integrado que permite ver el documento en cualquier dispositivo. Los desarrolladores pueden convertir formatos fácilmente con solo dos líneas de código.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Guardar documento en diferentes formatos - Java
     </h3>
     <pre><code class="java">// carga el archivo a convertir

Document doc = new Document(SD_PATH + "template.doc");

// Convertir DOC a DOCX, PDF y HTML

doc.save(SD_PATH + "output.docx", SaveFormat.DOCX);

doc.save(SD_PATH + "output.pdf", SaveFormat.PDF);

doc.save(SD_PATH + "output.html", SaveFormat.HTML);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Compatibilidad con OOXML, Flat OPC y Word 2003 XML
    </h2>
    <p>
     Esta biblioteca proporciona una amplia compatibilidad para cargar, ver, guardar y convertir archivos de WordprocessingML. Una implementación tan completa es posible porque la API se diseñó teniendo en cuenta la estructura de los documentos de Microsoft Word (y se sabe que WordprocessingML imita la representación interna de los archivos de Microsoft Word), lo que nos facilita incluir soporte para estas implementaciones en nuestro motor de producto principal.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Cargar y guardar formatos HTML
    </h2>
    <p>
     La importación de HTML y XHTML de las API de Android Word es resistente a los documentos que no están bien formados. Los elementos o atributos no válidos o no reconocidos se ignoran de forma segura durante el proceso de importación, mientras que la conversión a formatos HTML y XHTML es personalizable. Se pueden especificar muchos parámetros, como la exportación de propiedades de documentos, encabezados y pies de página, ubicaciones de imágenes, escala y resolución, opciones de exportación de campos de formulario y otros.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Manipulación con el modelo de objeto del documento
    </h2>
    <p>
     La biblioteca de documentos de Android permite a los desarrolladores crear, compilar y modificar archivos, incluido el formateo con un modelo de objeto de documento intuitivo que consta de más de 100 clases.
    </p>
    <p>
     Los desarrolladores de Android pueden crear, modificar, extraer y reemplazar mediante programación todos los elementos, incluidas secciones, encabezados, pies de página, párrafos, listas, tablas, texto, campos, hipervínculos, marcadores e imágenes. La API también permite especificar un formato detallado para objetos de dibujo, incluidas formas, cuadros de texto, imágenes, objetos OLE y controles.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Informes y combinación de correspondencia
    </h2>
    <p>
     Puede utilizar la API de Microsoft Word de Android como una solución de generación de informes con todas las funciones. Diseñe informes en Microsoft Word y luego permita que API complete documentos con datos de varias fuentes.
    </p>
    <p>
     La API permite a los desarrolladores utilizar campos de combinación de correspondencia estándar de Microsoft Word en los informes. Además, puede usar la sintaxis extendida de las API para los campos de combinación de correspondencia para realizar operaciones más complejas durante las operaciones de combinación de correspondencia, como insertar imágenes o regiones repetibles. Los datos que impulsan sus informes y fusiones de correo pueden provenir de objetos ResultSet, objetos comerciales o de una fuente de datos completamente personalizada.
    </p>
