<p>
 Aspose.Words for Reporting Services es una extensión de representación que permite a los desarrolladores exportar informes RDL y RDLC como archivos DOC, DOCX, PDF, RTF, HTML, MHTML, ODT, TXT, XPS y WordprocessingML desde Microsoft SQL Server Reporting Services.
</p>

<p>
 Aspose.Words for Reporting Services es totalmente compatible con las especificaciones RDL. Puede usar cualquier diseñador de informes para rediseñar sus informes existentes o simplemente exportar cualquier informe RDL existente como un documento de Microsoft Word, y se representará exactamente como lo esperaría: idéntico al diseño RDL.
</p>

    <h2 class="h2title">
     Alta fidelidad, idéntico al diseño de informes
    </h2>
    <p>
     Aspose.Words for Reporting Services exporta informes a archivos de Microsoft Word de manera que parezcan idénticos al diseño del informe original producido por los procesadores integrados de Microsoft SQL Server Reporting Services. Puede elegir entre dos modos de exportación o probar ambos tipos de diseño cambiando un solo ajuste de configuración para descubrir cuál se adapta mejor a sus necesidades:
    </p>
    <ul class="unstyled">
     <li>
      Diseño de flujo
     </li>
     <li>
      Diseño posicionado
     </li>
    </ul>
    <br/>
    <p>
     <strong>
      Diseño de flujo
     </strong>
     produce "editable" documentos que constan de párrafos simples, tablas, secciones: esto es similar en comparación con un empleado de oficina que crea el documento en Microsoft Word.
    </p>
    <p>
     <strong>
      Diseño posicionado
     </strong>
     produce informes con la fidelidad de conversión más precisa de RDL a Microsoft Word. Las tablas y matrices de los informes se exportan como tablas de Microsoft Word, mientras que algunos elementos se exportan como cuadros de texto posicionados para mantener la fidelidad lo más cerca posible del diseño original.
    </p>
   </div>
   <div class="col-lg-12">

     Automatización de Microsoft Word: no es necesario
    </h2>

Es una alternativa perfecta de automatización de Microsoft Word en términos de funciones compatibles, seguridad, estabilidad, escalabilidad, velocidad y precio.
