<p>
 Aspose.Words for Reporting Services je rozšíření pro vykreslování, které vývojářům umožňuje exportovat sestavy RDL a RDLC jako soubory DOC, DOCX, PDF, RTF, HTML, MHTML, ODT, TXT, XPS a WordprocessingML ze služby Microsoft SQL Server Reporting Services.
</p>

<p>
 Aspose.Words for Reporting Services plně podporuje specifikace RDL. K přepracování stávajících sestav můžete použít libovolného návrháře sestav nebo jednoduše exportovat jakoukoli existující sestavu RDL jako dokument Microsoft Word a vykreslí se přesně tak, jak byste to očekávali – identické s návrhem RDL.
</p>

Můžete si vybrat mezi dvěma režimy exportu nebo vyzkoušet oba typy rozložení změnou jediného konfiguračního nastavení, abyste zjistili, který z nich nejlépe vyhovuje vašim potřebám:

    </p>
    <p>
     <strong>
      Umístěné rozvržení
     </strong>
     vytváří sestavy s nejpřesnější věrností převodu z RDL do Microsoft Word. Tabulky a matice ze sestav jsou exportovány jako tabulky Microsoft Word, zatímco některé prvky jsou exportovány jako umístěná textová pole, aby se zachovala věrnost co nejblíže původnímu návrhu.

Jedná se o dokonalou alternativu automatizace Microsoft Word z hlediska podporovaných funkcí, zabezpečení, stability, škálovatelnosti, rychlosti a ceny.
