<p><strong>Aspose.Words for Python via .NET</strong> je balíček určený ke čtení a manipulaci s dokumenty různých typů, s primárním zaměřením na dokumenty Microsoft Word (DOC, DOCX, ODT), PDF a webové (HTML, Markdown) dokumenty. Naše knihovna obsahuje více než 100 tříd Pythonu pro zpracování nízkoúrovňových operací za zpracováním dokumentů a formátováním dat. Aspose.Words tedy umožňuje vývojářům Pythonu jedinečnou příležitost implementovat automatizaci dokumentů založenou na skriptech.</p>
<p>S příchodem výkonného počítačového hardwaru v posledních desetiletích se Python stal programovacím jazykem široce používaným při vývoji webu (rámce Django, Flask, Pyramid), vědeckých úlohách a analýze dat. Užitečnost Pythonu vychází z rozsáhlého ekosystému balíčků třetích stran. Python je dodáván s velkým množstvím progresivně se vyvíjejících knihoven a je považován za nejlepší volbu pro rychlý vývoj softwaru.</p>
<p>Tým Aspose.Words se rozhodl držet krok s nejnovějšími trendy v programování a dodávat komunitě Python profesionální software pro zpracování DOC, DOCX, PDF, RTF, DOT, DOTX, DOTM, DOCM, FlatOPC, ODT, OTT, HTML, MHTML, MOBI, TXT, XPS, PS, TIFF, JPEG, PNG, BMP, SVG, EMF, GIF, PCL, EPUB, XML, XAML a další formáty souborů. Můžeš vidět <a href="https://docs.aspose.com/words/python-net/supported-document-formats/">úplný seznam podporovaných formátů</a>.</p>
<p><i>Poznámka:</i> podle TIOBE Programming Index, který odráží popularitu programovacích jazyků, je dnes Python programovacím jazykem číslo jedna.</p>
<h2 class="h2title">Funkčnost Aspose.Words pro Python přes .NET</h2>
<p>Aspose.Words je samostatná knihovna Pythonu. Aby váš Python kód fungoval, nemusíte instalovat žádný kancelářský balík. Toto Python API se spoléhá na <a href="https://docs.aspose.com/words/python-net/aspose-words-document-object-model/">Objektový model dokumentu (DOM)</a> pro přístup k uzlům a prvkům dokumentu.</p>

      Vytvořte dokument a uložte jej do PDF – Python
     </h3>

import aspose.words as aw

doc = aw.Document()
