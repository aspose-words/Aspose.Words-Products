<p><strong>Aspose.Words for C++</strong> 是一种高级 Word 文档处理 API，用于执行各种处理任务。 Words 的 API 直接在您自己的 C++ 应用程序中提供文档格式化、操作、邮件合并、水印和更多功能，而无需 Microsoft Word。</p>
<p>Aspose.Words API 支持大多数 Microsoft Word 格式，并且可以将文档页面呈现为多种图像格式。因此，您可以使用 DOC、DOCX、PDF、RTF、DOT、DOTX、DOTM、DOCM、FlatOPC、ODT、OTT、HTML、MHTML、MOBI、TXT、XPS、PS、TIFF、JPEG、PNG、BMP、 SVG、EMF、GIF、PCL、EPUB、XML、XAML 等。 你可以看到 <a href="https://docs.aspose.com/words/cpp/supported-document-formats/">支持格式的完整列表</a>.</p>
<h2 class="h2title">Functionality of Aspose.Words for C++</h2>
<p>Aspose.Words 是一个独立的 C++ 库。您无需安装其他应用程序即可使您的 C++ 代码正常工作。 这个 C++ API 依赖于 <a href="https://docs.aspose.com/words/cpp/aspose-words-document-object-model/">文档对象模型 (DOM)</a> 访问文档节点和元素。</p>
