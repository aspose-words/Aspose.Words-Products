<p>
 Aspose.Words for Android via Java 是一个文档处理 API，使您能够直接在 Android 应用程序中执行文档操作任务。 API 允许您轻松创建、操作、转换和呈现 Microsoft Word 文件。
</p>

<p>
 Aspose.Words for Android via Java 支持处理原生格式，如 DOC、DOCX、OOXML、TXT、RTF 以及许多其他常用格式，如 HTML、OpenDocument ODT、PDF、EPUB、XPS 和图像格式，如 JPEG、PNG 、SVG 和 EMF。
</p>

    <h2 class="h2title">
     文件格式转换
    </h2>
    <p>
     能够以高精度快速可靠地转换不同文件是我们大多数客户选择 API 的原因。 Android Microsoft Word API 允许您将文件从任何支持的格式转换为任何其他支持的格式，例如将任何文档转换为高保真 SWF，生成的 SWF 包含文档页面和嵌入式查看器，允许在任何设备上查看文档。开发人员只需两行代码即可轻松转换格式。

// 将 DOC 转换为 DOCX、PDF 和 HTML

doc.save(SD_PATH + "output.pdf", SaveFormat.PDF);

doc.save(SD_PATH + "output.html", SaveFormat.HTML);</code></pre>

    <p>
     Android 开发人员可以通过编程方式创建、修改、提取和替换所有元素，包括部分、页眉、页脚、段落、列表、表格、文本、字段、超链接、书签和图像。 API 还允许为绘图对象指定详细的格式，包括形状、文本框、图像、OLE 对象和控件。
